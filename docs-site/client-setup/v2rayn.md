# v2rayN 配置（Windows）

1. 从 [官方发布页](https://github.com/2dust/v2rayN/releases) 下载 v2rayN。
2. 解压并运行 `v2rayN.exe`。
3. 在主窗口中，点击 **服务器** → **从剪贴板导入批量 URL** 或 **从 URL 导入**。
4. 粘贴 V2Ray 订阅 URL：
   ```
   https://raw.githubusercontent.com/MS33834/ProxieHub/main/nodes/v2ray.txt
   ```
5. 如有需要，点击 **更新订阅**。
6. 右键点击一个节点并选择 **设为活动服务器**。
7. 点击 **系统代理** → **自动配置系统代理**。

## 提示

- 如果某个节点无法连接，尝试换一个。免费节点失效很快。
- 保持 v2rayN 更新，以支持最新的协议。
