# Shadowrocket 配置（iOS）

1. 在 App Store 购买并安装 Shadowrocket。
2. 打开应用，点击 **添加节点**。
3. 类型选择 **Subscribe**。
4. 粘贴订阅 URL：
   ```
   https://raw.githubusercontent.com/MS33834/ProxieHub/main/nodes/v2ray.txt
   ```
5. 点击 **保存**。
6. 选择一个节点并打开连接开关。

## 提示

- Shadowrocket 是付费应用，没有官方免费版。
- 如果订阅失败，尝试在 Wi-Fi 和移动数据之间切换。
