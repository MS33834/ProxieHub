# Shadowrocket 配置教程（iOS）

1. 从 App Store 购买并安装 Shadowrocket。
2. 打开应用，点击右上角 **+** 添加节点。
3. 选择 **类型** → **Subscribe**。
4. 粘贴订阅链接：
   ```
   https://raw.githubusercontent.com/MS33834/ProxieHub/main/nodes/v2ray.txt
   ```
5. 点击 **保存**。
6. 选择一个节点，打开连接开关。

## 使用建议

- Shadowrocket 为付费应用，没有官方免费版本。
- 如果订阅更新失败，尝试切换 Wi-Fi 与蜂窝数据。
