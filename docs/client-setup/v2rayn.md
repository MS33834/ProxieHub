# v2rayN Setup (Windows)

1. Download v2rayN from the [official releases](https://github.com/2dust/v2rayN/releases).
2. Extract and run `v2rayN.exe`.
3. In the main window, click **Servers** → **Import batch from clipboard** or **Import from URL**.
4. Paste the V2Ray subscription URL:
   ```
   https://raw.githubusercontent.com/MS33834/ProxieHub/main/nodes/v2ray.txt
   ```
5. Click **Update subscription** if available.
6. Right-click a node and select **Set as active server**.
7. Click **System Proxy** → **Auto config system proxy**.

## Tips

- If a node does not connect, try another. Free nodes die quickly.
- Keep v2rayN updated to support the latest protocols.
