# v2rayNG 配置教程（Android）

1. 从 GitHub releases 或可信来源安装 v2rayNG。
2. 打开应用，点击右下角 **+**。
3. 选择 **从 URL 导入配置**。
4. 粘贴 V2Ray 订阅链接：
   ```
   https://raw.githubusercontent.com/MS33834/ProxieHub/main/nodes/v2ray.txt
   ```
5. 点击 **确定** 下载订阅。
6. 点击一个节点选择它。
7. 点击底部 **V** 图标连接。

## 使用建议

- 提示时授予 VPN 权限。
- 使用内置延迟测试找到相对更快的节点。
