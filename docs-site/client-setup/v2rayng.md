# v2rayNG 配置（Android）

1. 从 GitHub Releases 或可信来源安装 v2rayNG。
2. 打开应用，点击 **+** 按钮。
3. 选择 **从 URL 导入配置**。
4. 粘贴 V2Ray 订阅 URL：
   ```
   https://raw.githubusercontent.com/MS33834/ProxieHub/main/nodes/v2ray.txt
   ```
5. 点击 **确定** 下载订阅。
6. 点击一个节点以选中它。
7. 点击底部的 **V** 图标连接。

## 提示

- 提示时授予 VPN 权限。
- 使用内置延迟测试寻找更快的节点。
